<p>After hours of googling I found the answer myself..
there is already a question in SO <a href="https://stackoverflow.com/questions/4646147/is-there-open-source-code-for-making-link-preview-text-and-icons-like-in-face">Is there open-source code for making 'link preview' text and icons, like in facebook?</a> . So we can use this link <a href="http://api.embed.ly/1/oembed?url=YOUR-URL" rel="nofollow noreferrer">http://api.embed.ly/1/oembed?url=YOUR-URL</a> via http GET where we get the meta tags in json format..
I wrote my own code using <a href="https://github.com/tmpvar/jsdom" rel="nofollow noreferrer">JSdom</a> and here goes the code...</p>

<p>Server side code:</p>

<pre><code>app.post( '/scrapUrl/', function( req, res ) {

    var jsdom = require( 'jsdom' );
    var jsonRes = {};
    jsdom.env( {
        url: req.body.url,
        scripts: [ "http://code.jquery.com/jquery.js" ],
        done: function( error, window ) {
          var $ = window.$;

          $( 'meta' ).each( function() {
            var name = $( this ).attr( 'property' );
            var value = $( this ).attr( 'content' );
            if ( name ) {
              jsonRes[ name.slice( 3 ) ] = value;
              console.log( name + ": " + value );
            }
          } );
          res.send( jsonRes );
        }
    } );
} );
</code></pre>

<p>and the angular code</p>

<pre><code>$http.post( '/scrapUrl/', {
    url: url  //send the url U need to scrape
} ).then( function( res ) {
    console.log(res.data)//U get the meta tags as json object
});
</code></pre>

<p>Once you get the JSON object you can display it in whatever style you want.</p>
